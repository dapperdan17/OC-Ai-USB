=============================================
  Portable OpenCode AI USB
  Version 1.18.3
=============================================

SELF-CONTAINED AI CODING ASSISTANT
Works on any Windows 10/11 PC. No installation needed.

HOW TO USE:
  1. Plug this USB into any Windows PC
  2. Open the USB drive in File Explorer
  3. Double-click "OpenCode AI.exe"
  4. OpenCode starts in a terminal window
  5. When you exit, your session log is saved

  (First run on a new PC: if Windows SmartScreen warns about an
   unknown app, choose "More info" then "Run anyway".)

NO API KEY NEEDED TO START:
  OpenCode includes a free built-in model, so you can start
  chatting straight away. To use your own AI provider, type
  /connect inside OpenCode, or edit data\config\opencode.json.

  Supported: OpenAI, Anthropic (Claude), Google Gemini,
  GitHub Copilot, Ollama (local), and 70+ more.

SESSION LOGGING:
  Every session is automatically logged to:
    sessions\YYYY-MM-DD_HH-MM-SS\session.md

  Each log contains the date/time, the conversation, and the
  tasks completed. Browse them via the "Session Logs.bat" shortcut.

WHAT'S INCLUDED:
  - OpenCode v1.18.3 (terminal AI coding agent)
  - WezTerm portable terminal (GPU-accelerated)
  - Node.js v26.5.0 portable runtime
  - ripgrep (bundled - no download needed)

REQUIREMENTS:
  - Windows 10/11 (64-bit)
  - USB 3.0 port (recommended)
  - Internet connection only for cloud AI providers
  - ~500MB free space

DRIVE LAYOUT (varies by USB size and installer settings):
  Tools partition: OpenCode program and tools
  Storage partition: your projects and files (if you chose two)

  Drive letters are assigned automatically by Windows and will
  vary between computers. Look for the "OpenCode AI" volume label
  to identify the tools drive. Everything on the stick locates
  itself automatically, so the drive letter does not matter.

YOUR DATA STAYS ON THE STICK:
  Credentials, session history, cache and temp files are all kept
  on this USB drive. The app writes nothing into the host
  computer's user profile.

  (Note: Windows itself always records that a USB device was
  connected, in its own logs - that is outside any app's control.
  What is guaranteed is that your files, keys and conversations
  never touch the host computer's disk.)
